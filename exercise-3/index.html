Exercise 3 – Form Validation

Open script.js in your editor and follow the instructions in each section below. Try submitting the form with different inputs to test your validation logic.

Exercise 1: Basic Form Validation

Required Fields – In script.js, listen for the form's submit event. If the Name or Email field is empty, call event.preventDefault() to stop submission and show an alert.
Email Format – Extend your validation to check that the email field contains a valid email address using a regular expression: /^[^\s@]+@[^\s@]+\.[^\s@]+$/

Challenge Tasks

Password Confirmation – Add validation to ensure the Password and Confirm Password fields match before the form is submitted.
Real-time Email Validation – In script.js, add an input event listener on the Email field. As the user types, display feedback in #emailFeedback: set its text to 'Valid email address.' (green) or 'Invalid email address.' (red) based on the regex test.
