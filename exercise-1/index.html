Select by Tag Name – Use document.getElementsByTagName('p') to select all <p> elements and change their style.color to 'red'.
Select by Query Selector – Use document.querySelector('.highlight') to select the first highlighted paragraph and set its style.border to '1px solid black'. Then use document.querySelector('.container p') to select the first paragraph inside the container and set its style.fontStyle to 'italic'.
