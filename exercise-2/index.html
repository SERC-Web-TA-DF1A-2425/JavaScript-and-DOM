Exercise 1: Inline Event Handlers

Example: <button onclick="document.getElementById('message').innerText = 'Button Clicked!'">

Mouse Over Event (inline) – Add an onmouseover attribute to the div below so that hovering over it changes its background color to yellow.

Exercise 2: Declared Event Handlers

Click Event – In script.js, use addEventListener('click', ...) on #myButton to update #message when the button above is clicked.
